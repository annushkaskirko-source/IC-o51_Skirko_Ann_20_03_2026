## Мета роботи
Завершити розробку програмного застосунку та підготувати його до підсумкової перевірки.

## Завдання роботи
1. Виконати рефакторинг програмного коду.  
2. Підготувати програмний застосунок до фінальної демонстрації.  
3. Додати базові модульні тести.  
4. Підготувати документацію програмного застосунку.  

## Використані технології та підходи
- **Мова програмування:** C#  
- **Рефакторинг:** Оптимізація коду, розділення класів, покращення читаності  
- **Модульне тестування:** NUnit, xUnit або MSTest  
- **Документація:** README, XML-коментарі, блоки прикладів  

## Опис фінального функціоналу
- Графічний інтерфейс для CRUD-операцій  
- Персистентність даних у JSON  
- Валідація введених даних та бізнес-правила  
- Модульні тести для перевірки основної логіки
